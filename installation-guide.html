1

Λήψη για Windows

Το POSPal είναι διαθέσιμο για Windows 10/11 (64-bit):

Windows

Windows 10/11 (64-bit)

Λήψη για Windows
2

Λήψη και εγκατάσταση

Μετά τη λήψη του αρχείου:

  • Windows: Εκτελέστε το .exe αρχείο και ακολουθήστε τον οδηγό εγκατάστασης
Σημαντικό: Για Windows, ενδέχεται να χρειαστεί να απενεργοποιήσετε την προστασία από ιούς ή να επιτρέψετε την εφαρμογή από το Windows Defender.
3

Πρώτη εκκίνηση

Όταν εκκινήσετε το POSPal για πρώτη φορά:

  • Θα σας ζητηθεί να δημιουργήσετε λογαριασμό διαχειριστή (αυτό είναι απαραίτητο για να μπορεί το PDA mobile version να συνδέεται με τον server μέσω του firewall)
  • Ρυθμίστε τις προτιμήσεις εκτύπωσης και γλώσσας
  • Δημιουργήστε το πρώτο σας μενού
4

Ρύθμιση συσκευών

Για να συνδέσετε εκτυπωτές και άλλες συσκευές:

  • Πηγαίνετε στις Ρυθμίσεις → Συσκευές
  • Επιλέξτε τον εκτυπωτή σας από τη λίστα
  • Κάντε δοκιμή εκτύπωσης για να επιβεβαιώσετε τη σύνδεση
  • Ρυθμίστε το μέγεθος του χαρτιού και την ποιότητα εκτύπωσης

5

Επιβεβαίωση εγκατάστασης

Για να επιβεβαιώσετε ότι η εγκατάσταση ήταν επιτυχής:

  • Εκκινήστε το POSPal
  • Δημιουργήστε μια δοκιμαστική παραγγελία
  • Εκτυπώστε μια απόδειξη
  • Ελέγξτε ότι όλες οι λειτουργίες λειτουργούν σωστά
Βοήθεια: Αν αντιμετωπίσετε προβλήματα κατά την εγκατάσταση, επικοινωνήστε μαζί μας στο Instagram @pospalsoftware.
6

Επόμενα βήματα

Μετά την επιτυχημένη εγκατάσταση:

  • Δημιουργήστε τα μενού σας
  • Ρυθμίστε τις τιμές και τις κατηγορίες
  • Προσθέστε τους υπαλλήλους σας
  • Ρυθμίστε τις αναφορές και τα στατιστικά
  • Εκπαιδεύστε το προσωπικό σας
